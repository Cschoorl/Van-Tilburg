## Bestand

| Bestand           | Beschrijving                         |
|------------------|--------------------------------------|

## Lokaal bekijken

1. Dubbelklik `van-tilburg.html` of sleep het naar Chrome / Safari / Firefox.
2. Of vanuit deze map:

   ```bash
   open van-tilburg.html
   ```

## Presenteren

- **Volgende slide:** pijl rechts, spatie of Page Down  
- **Vorige slide:** pijl links of Page Up  
- **Direct naar slide 1–8:** toetsen `1` t/m `8`  
- **Mobiel:** swipe links/rechts  

## PDF exporteren

Browser: **Print → Opslaan als PDF** (print-stylesheets zetten elke slide op een aparte pagina).
